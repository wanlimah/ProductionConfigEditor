TECHNICAL DETAILS
═══════════════════════════════════════════════════════════════════

Old Logic:
  if (record.Status == "FAIL")
  
  Problem: Exact match only
  "FAIL" matches → RED ✅
  "FAILED" doesn't match → BLACK ❌

New Logic:
  var statusUpper = record.Status?.ToUpper() ?? "";
  if (statusUpper.Contains("FAIL"))
  
  Solution: Contains check (case-insensitive)
  "FAIL" contains "FAIL" → RED ✅
  "FAILED" contains "FAIL" → RED ✅
  "Failed" contains "FAIL" → RED ✅
  "fail" contains "FAIL" → RED ✅
