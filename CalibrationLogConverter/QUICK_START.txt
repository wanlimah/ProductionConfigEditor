═══════════════════════════════════════════════════════════════════════
  🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

  PROBLEM: "No suitable parser found"
  SOLUTION: File doesn't match known vendor patterns.
            Rename to include: "broadcom", "fm-002", or "logsheet"

  PROBLEM: Missing data in preview
  SOLUTION: Check source Excel file has recognizable column names
            (Model, Serial Number, Due Date)

  PROBLEM: Dates showing as numbers
  SOLUTION: In source file, format date column as Date (not Text)

  PROBLEM: Can't find files
  SOLUTION: Use Browse button to select files manually
