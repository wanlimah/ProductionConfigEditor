═══════════════════════════════════════════════════════════════════
FILTERING LOGIC (IMPROVED)
═══════════════════════════════════════════════════════════════════

FOR "ON-SITE UPDATE TEMPLATE" WORKSHEET:

Step 1: Check if row is a HEADER
   → Contains "Model" AND "SN"?
   → YES: Detect columns, increment skippedRows, CONTINUE (skip)

Step 2: Check if row is PAGE BREAK
   → Contains "Company Confidential"?
   → Contains "Confidential"?
   → Contains "Page" AND "of"?
   → YES: Increment skippedRows, set inDataSection=false, CONTINUE (skip)

Step 3: Check if in data section
   → Not in data section? → CONTINUE (skip)
   → Columns not detected? → CONTINUE (skip)

Step 4: Process data row
   → Extract Model, Serial, Status, Cal Date
   → Skip if empty/invalid
   → Add to results
