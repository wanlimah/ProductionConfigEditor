Possible Reason 1: Column Header Name Mismatch
───────────────────────────────────────────────
Your column header might be named differently than expected.

Expected Names for CALIBRATION DATE (Column D):
   • "Date dd-mm-yy"
   • "Date (dd-mm-yy)"
   • "Date"
   • "Calibration Date"
   • "Cal Date"
   • etc.

Expected Names for CAT INT (Column I):
   • "Cat Int Month"
   • "Cat Int (Month)"
   • "Interval"
   • "Cal Interval"
   • "Months"
   • etc.

→ The popup will show me your ACTUAL column names!

Possible Reason 2: Column Data Format
──────────────────────────────────────
Even if headers are found:
   • Date column might contain text instead of dates
   • Cat Int column might contain text instead of numbers
   • Values might be empty

→ Debug output will show this too!
