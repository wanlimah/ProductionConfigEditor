═══════════════════════════════════════════════════════════════════
HOW IT WORKS
═══════════════════════════════════════════════════════════════════

For each row in the worksheet:

1. CHECK: Is this a header row?
   → Look for "Model" AND "SN" in the same row
   → If YES: Detect column positions for this section
   → Set "in data section" flag to TRUE

2. CHECK: Is this a page break row?
   → Look for "Company Confidential", "Page X of Y"
   → If YES: Set "in data section" flag to FALSE
   → Wait for next header row

3. CHECK: Are we in a data section?
   → If NO: Skip this row
   → If YES: Extract calibration data using current column mapping

4. EXTRACT: Read data from columns
   → Model, Serial Number, Status, Cal Date
   → Calculate Due Date = Cal Date + 12 months
   → Add to results

5. REPEAT for ALL rows in the file

Result: ALL calibration data from ALL sections extracted! ✅
