═══════════════════════════════════════════════════════════════════
FILTERING LOGIC
═══════════════════════════════════════════════════════════════════

The parser now checks each row and skips if:

1. Model = "X" OR Serial = "X"
   → Skip (error marker)

2. Model is empty AND Serial is empty
   → Skip (blank row)

3. Model = "Model" OR Model = "Model Number"
   → Skip (repeated header)

4. Serial = "SN" OR Serial = "Serial Number"
   → Skip (repeated header)

5. Model contains "Company Confidential"
   → Skip (page break)

6. Serial contains "Company Confidential"
   → Skip (page break)

Only extract rows with REAL equipment data!
