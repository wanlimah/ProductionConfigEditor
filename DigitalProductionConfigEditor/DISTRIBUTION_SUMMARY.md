## 🔄 Multi-Target Configuration

The application is configured to support **both .NET 6.0 and .NET 8.0** from a single codebase.

**Project File Setting:**
```xml
<TargetFrameworks>net6.0-windows;net8.0-windows</TargetFrameworks>
```

This means:
- ✅ One codebase, two outputs
- ✅ Simultaneous building
- ✅ Easy to add future .NET versions
- ✅ No code duplication
