╚══════════════════════════════════════════════════════════════════════════╝

Common Mistakes (BEFORE with text boxes)
─────────────────────────────────────────────────────────

❌ enable: [TURE    ]  ← Typo!
❌ enable: [true    ]  ← Wrong case!
❌ enable: [1       ]  ← Wrong value!
❌ mode:   [auto    ]  ← Wrong case!
❌ mode:   [AUTOO   ]  ← Typo!
❌ rule:   [DATE    ]  ← Invalid option!

Result: XML with incorrect values, may cause errors at runtime


No Mistakes Possible (AFTER with dropdowns)
─────────────────────────────────────────────────────────

✅ enable: [TRUE   ▼]  ← Only TRUE or FALSE available
✅ enable: [FALSE  ▼]  ← Always correct case
✅ mode:   [AUTO   ▼]  ← Only valid options shown
✅ rule:   [REV    ▼]  ← Can't type invalid value

Result: XML always has correct values, no runtime errors!
