THE PCB CONFUSION PROBLEM (SOLVED)
================================================================================

❌ BEFORE: Users tried to add NEW PCB Format Config for each product
   - Clicked "➕ Add PCB Format Config" multiple times
   - Created confusion and incorrect structure
   - PCB only allows ONE instance per XML

✅ AFTER: Clear guidance at multiple touchpoints
   - Startup guide explains the difference
   - Visual warnings prevent mistakes
   - Tooltips provide just-in-time help
   - Help button for on-demand reference

KEY MESSAGE: "Add PCB ONCE, Edit to add islands (products)"

================================================================================
TWO WORKFLOWS CLEARLY EXPLAINED
================================================================================

🆕 WORKFLOW 1: Adding PCB (First Time)
   Step 1: Find "🔲 PCB Format Config" section in left panel
   Step 2: Click "➕ Add PCB Format Config"
   Step 3: PCB structure appears in right panel
   ⚠️ Do this ONLY ONCE per XML file!

✏️  WORKFLOW 2: Editing PCB (Most Common)
   Step 1: Find "PCB Format Config" in right panel
   Step 2: Click "✏ Edit" button
   Step 3: Add/Edit/Delete islands in dialog
   Step 4: Click "💾 Save"
   
   Each product = Different island inside same PCB!
