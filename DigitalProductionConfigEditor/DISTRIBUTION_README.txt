Thank you for downloading Digital Production Config Editor!

--------------------------------------------------------------------------------
  QUICK START GUIDE
--------------------------------------------------------------------------------

1. EXTRACT ALL FILES
   - Extract this ZIP file to a folder on your computer
   - Keep all files together in the same folder

2. INSTALL .NET RUNTIME (if needed)
   
   FOR .NET 6.0 VERSION:
   - Visit: https://dotnet.microsoft.com/download/dotnet/6.0
   - Download: ".NET Desktop Runtime 6.0.x" (x64)
   - Install and restart
   
   FOR .NET 8.0 VERSION:
   - Visit: https://dotnet.microsoft.com/download/dotnet/8.0
   - Download: ".NET Desktop Runtime 8.0.x" (x64)
   - Install and restart

3. RUN THE APPLICATION
   - Double-click: DigitalProductionConfigEditor.exe
   - If you see a security warning, click "More info" then "Run anyway"
   - The application should launch!

--------------------------------------------------------------------------------
  WHICH VERSION DO I HAVE?
--------------------------------------------------------------------------------

Check the ZIP filename you downloaded:
- DigitalProductionConfigEditor_v2.0_NET6.zip = .NET 6.0 version
- DigitalProductionConfigEditor_v2.0_NET8.zip = .NET 8.0 version

Not sure which .NET you have installed?
- Open PowerShell
- Type: dotnet --list-runtimes
- Look for: Microsoft.WindowsDesktop.App 6.x.x or 8.x.x

--------------------------------------------------------------------------------
  SYSTEM REQUIREMENTS
--------------------------------------------------------------------------------

- Windows 10 (64-bit) or later
- .NET 6.0 OR .NET 8.0 Desktop Runtime (see above)
- 512 MB RAM minimum
- 50 MB free disk space
- 1024x768 screen resolution or higher

--------------------------------------------------------------------------------
  FIRST-TIME USAGE
--------------------------------------------------------------------------------

When you launch the application:

1. You'll see two panels:
   - LEFT: Master Template (all available configurations)
   - RIGHT: Your New Configuration (empty at first)

2. Click [+ Add] next to configurations you want

3. Click [Next →] to edit packages

4. Click [Next →] to review

5. Click [💾 Save] to save your configuration file

For detailed instructions, open: README_NEW_WORKFLOW.md

--------------------------------------------------------------------------------
  TROUBLESHOOTING
--------------------------------------------------------------------------------

APPLICATION WON'T START?
→ Install the correct .NET Runtime (see above)
→ Make sure Master_Digital_ProductionUserConfig.xml is in the same folder
→ Try "Run as Administrator"

"MASTER XML NOT FOUND" ERROR?
→ Re-extract the ZIP file
→ Don't move files separately - keep them together

STILL HAVING ISSUES?
→ Read INSTALLATION_GUIDE.md for detailed troubleshooting
→ Check that you're using Windows 10 or later
